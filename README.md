# Inventory & Loan System

## Project description

A simple inventory and equipment loan system for managing devices, users, and borrowing history.

## Technologies

* PHP
* MySQL (XAMPP)
* HTML/CSS
* GitHub

## Features

* Add and manage devices
* Borrow and return equipment
* Track loan history
* Search and filter devices

## Team

* Roman: UI, documentation, GitHub management, CSS
* Artem: backend (PHP + MySQL)

## How to run

1. Install XAMPP
2. Import database
3. Start Apache & MySQL
4. Open `localhost/project-folder`
